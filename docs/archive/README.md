# docs/archive

Documentos de fases completadas. Se archivan en lugar de eliminarse para preservar
el contexto historico: decisiones de diseno, razonamiento, y deuda tecnica aceptada.

## Convencion

Cuando una fase termina:

1. Su roadmap detallado (`PHASE_X.Y_*.md`) se mueve aqui.
2. `docs/ROADMAP.md` se actualiza con un resumen compacto de la fase.
3. Las decisiones de diseno permanentes se migran a `docs/ARCHITECTURE.md` o `docs/SECURITY.md`.
